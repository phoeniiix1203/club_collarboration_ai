# Galaxy2Painting

## Running locally for development

### Install

```bash
yarn run prep
```

If installation failed please check your NodeJS version, this project works on the latest NodeJS version(19.1.0)

### Test

```bash
yarn run start
```

And go to `localhost:9966`
